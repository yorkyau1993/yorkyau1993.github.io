🛡️ 全民防衛生存測驗 - 更新

Ver 1.6 - 教學功能導入
新增解析功能：答題後顯示「老師補充」與正確答案。

手動切換：取消自動跳轉，改為手動按下「下一題」，方便課堂講解。

Ver 2.0 - 內容深化與警示 (Content Update)
題庫擴充：整合國防部《當危機來臨時》手冊內容。

嚴重後果警示：答錯時顯示紅色警示框，說明選錯的離譜下場（如：變成乾屍、被敵軍俘虜）。

文字優化：移除「老師補充：」字樣，直接顯示知識點。

Ver 3.0 - 多模式架構 (Multi-Mode)
三大模式確立：

個人練習版：手機直排、含解析。

極速挑戰版：手機直排、無解析（直接下一題）、測驗速度。

學校教學版：電腦橫向、含解析、成績結算。

團隊化設定：學校版新增「學校名稱」、「隊名」輸入欄位。

Ver 4.0 - 響應式與數據 (RWD & Data)
RWD 自適應佈局：

手機：維持直排單欄，字體放大好點擊。

電腦/平板：自動切換為橫向左右分割（左題目、右選項）。

CSV 存檔：學校模式新增「下載答題明細 (.csv)」功能，供老師課後分析。

獨立登入頁：學校模式改為獨立登入畫面，並增加「返回首頁」按鈕。

Ver 5.0 - 體驗優化 (UX Improvement)
移除 Alert：輸入錯誤改為頁面內紅字提示，不再彈出視窗。

版面防切：修正手機版內容被切掉的問題，優化頂部標題間距。

互動按鈕：新增「給自己掌聲」與「黨的回饋」互動按鈕。

Ver 6.0 - 視覺特效 (Visual Effects)
動畫導入：引入 canvas-confetti 函式庫。

成功特效：按下掌聲按鈕播放彩色煙火。

失敗特效：按下黨的回饋播放黃色星星與紅色紙花（諷刺效果）。

Ver 7.0 - 橫向滿版與社群 (Landscape Polish)
選項矩陣化：電腦版選項改為 2x2 滿版矩陣，視覺更整齊大氣。

社群連結：新增 Threads 追蹤按鈕。

下載優化：截圖功能增加邊距 (Padding)，避免文字貼邊。

Ver 8.0 - 戰報完美版 (Final Polish)
智慧截圖：下載戰報時，自動隱藏所有操作按鈕（如特效按鈕、重來按鈕），只保留感謝狀/證書內容。

證書樣式：調整感謝狀內距，使其更像一張正式的獎狀。

UI 微調：Threads 按鈕縮小並移至製作名單下方，保持畫面簡潔。
